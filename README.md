## Description
This app allows users to play livestreams (RTSP/HTTP) and add, edit, or remove custom text overlays on top of the video.

## Tech Stack
- React (Frontend)
- Flask (Backend)
- MongoDB (Database)
- RTSP / HTTP for video streaming

## Setup

### Backend
```
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend
```
cd frontend
npm install
npm start
```
